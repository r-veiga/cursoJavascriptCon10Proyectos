Conversión de String a Number en Javascript

Para evitar los problemas que surgen cuando se combina el tratamiento de números con Strings que contienen números, se aconseja hacer un buen uso de las conversiones entre tipos.
La conversión se usará mucho, cuando se recibe un valor de un formulario HTML, se recibe como String y para mis cálculos necesitaré transformarlo en número.

Los errores los encontramos cuando el operador tiene un uso sobrecargado.
No tenemos el problema con el - porque sólo se usa para restar, o con * para multiplicar .
Sí lo tenemos con el + porque puede usarse como suma o como concatenación, dependiendo.

Cuidado con el valor NaN "not a number". El valor NaN es de tipo Number.
Cuando hago una conversión con Number() o parseInt() de un String no numérico, obtengo NaN.
Pero NaN no es un error, así que arrastraré NaN hasta la siguiente operación matemática que realice. Y ahí romperá el programa.

  • Number( String )
  • parseInt( String )
  • parseFloat( String )
  • miNumber.toFixed( numDecimales ) :: String
