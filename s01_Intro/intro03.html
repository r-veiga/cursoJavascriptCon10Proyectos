Declarar variables con var

Y también el uso de const y let

var es una palabra para definir variables que está desactualizada.

let mejora a var y es la palabra indicada para definir variables.
No se permite emplear dos veces let en el mismo scope, lo que reduce errores.

const permite declarar constantes. Su valor no se puede reasignar.
La no reasignación de const es especialmente cierta para primitivas, pero en el caso de arrays y de objetos, aunque no se permite su reasignación sí es posible su modificación. Los arrays y objectos definidos como const son mutables.

El resultado de este ejercicio se puede ver en la consola con F12 (ejercicio en que se ha usado console.log())


        var nombre = 'Carlos';
        var apellido;
        console.log(nombre);                    // Carlos
        console.log(apellido);                  // undefined   
        
        var nombre = 'Juan';                    // VAR permite definir dos veces una variable en el mismo scope
        console.log(nombre);                    // Juan

        let coche = 'Audi';
        console.log(coche);

        let coche = 'Renault';                   // Uncaught SyntaxError: Identifier 'coche' has already been declared
        console.log(coche);                      // ningún código del script llegaría a ejecutarse por el fallo anterior

        const aeropuerto = 'Barajas';
        const arrayNumeros = [100, 200, 300];
        console.log(aeropuerto);                 // Barajas
        console.log(arrayNumeros);               // (3) [100, 200, 300]
        
        const aeropuerto = 'Barajas';            // Uncaught SyntaxError: Identifier 'aeropuerto' has already been declared
        
        arrayNumeros[0] = 7;
        arrayNumeros.push(8);
        console.log(arrayNumeros);               // (4) [7, 200, 300, 8]
        
        const mascota = { 
            raza: 'gato', 
            nombre: 'Manchitas' 
        };
        console.log(mascota);                    // {raza: "gato", nombre: "Manchitas"}
        mascota.raza = 'perro';
        mascota.edad = 4;                   
        console.log(mascota);                    // {raza: "perro", nombre: "Manchitas", edad: 4}
