Try / Catch / Finally en Javascript

Hay poco que explicar porque el funcionamiento es similar a Java.

La ejecución de un script Javascript se detiene cuando llega a un punto de error.
Como Javascript no es compilado, no me puede cantar error en compilación, se descubre en runtime. 


    try {
        invocacionQueVaAFallarPorInexistente();
    }
    catch( error ) {
        console.log(error);
        console.log(`Error recogido en CATCH, así que se continúa la ejecución del script`);
        console.log(`-- -- -- -- -- -- `);
    }
    finally {
        console.log(`Esto es bloque FINALLY que se ejecuta siempre`);
        console.log(`-- -- -- -- -- -- `);
    }
