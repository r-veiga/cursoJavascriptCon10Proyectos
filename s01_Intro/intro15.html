Funciones en Javascript

Las funciones se definen con la palabra function. Se distinguen tres tipos de funciones:

  1. funciones con nombre
    Una función que puede ser invocada por su nombre en cualquier momento. 
    2. 
        function saludar( nombre )        { console.log(" Hola " + nombre); }
        function devuelveSaludo( nombre ) { return "Hola " + nombre;        }
        function sumar( a, b )            { return a + b;                   }

        retorno1 = saludar("Arturo");          // retorno1 = undefined 
        retorno2 = devuelveSaludo("Adela");    // retorno2 = "Hola Adela" 
        let total = suma( 1, 4 )               // total = 5
        let otroTotal = suma ( 1, "4" );       // total = "14"
  2. funciones anónimas - una función que no tiene nombre.
    Como las funciones son objetos al fin y al cabo, la puedo asignar a una variable o a una propiedad de un objeto para luego invocarla empleando paréntesis ( ). 
    3. 
        let hola      = function( nombre ) { console.log(" Hola " + nombre); };
        let darHola   = function( nombre ) { return "Hola " + nombre; };
        let sumatorio = function( a, b )   { return a + b; };

        retorno1 = hola("Benito");               // retorno1 = undefined 
        retorno2 = darHola("Bárbara");           // retorno2 = "Hola Bárbara" 
        let total = sumatorio( 1, 4 )            // total = 5
        let otroTotal = sumatorio ( 1, "4" );    // total = "14"
  3. funciones IIFE (expresión de función inmediatamente invocada)
    Función ejecutada en el mismo punto que es declarada, mediante el uso de paréntesis ( ). 
    4. 
        ( function( altura ) {
            console.log('La altura de esta montaña es superior a ' + altura);
        }
        ( '8000 metros' ) );   // La altura de esta montaña es superior a 8000 metros
  4. métodos de propiedad
    Se definen como la propiedad de un objeto. Pueden definirse al tiempo que el objeto, o pueden definirse fuera. 
    5. 
        const musica = {
            reproducir: function( cancion ) { 
                            console.log(`Estoy reproduciendo la canción: ${cancion}`); 
                        }
        }   

        musica.cambiarCancion = function( otraCancion ) {
            console.log('Cambio a la canción: ' + otraCancion);
        }
        
        let retornoUndefined = musica.reproducir('Hey Jude');
        retornoUndefined = musica.reproducir('I am the walrus');

Controlar la recepción de un parámetro (default value)

Mientras que tiempo atrás se hacía con un if controlando que no estuviese undefined, actualmente se introduce un valor por defecto en la definición de la función. 


    function despedirALaAntigua( nombre ) {
        if( typeof nombre === 'undefined' ) { 
            let nombre = 'Visitante desconocido';
        }
        console.log('Adiós ${nombre}');
    }

    function despedirALaModerna( nombre = 'Visitante desconocido' ) {
        console.log('Adiós ${nombre}');
    }

    despedirALaAntigua();    // Adiós visitante desconocido
    despedirALaModerna();    // Adiós visitante desconocido


    function sumaDefaultVals( a = 3, b = 5 ) {
        return a + b;
    }

    let sumaDosSumandos  = sumaDefaultVals( 30, 10 );    // 40 = 30 + 10
    let sumaConUnDefault = sumaDefaultVals( 90 );        // 95 = 90 + (5)

High-order functions

Se denominan "high order functions" a las funciones que devuelven una función o reciben una función como parámetro.

return

Las funciones siempre devuelven un valor.
Cuando el valor devuelto no lo defino explícitamente, éste es undefined.
