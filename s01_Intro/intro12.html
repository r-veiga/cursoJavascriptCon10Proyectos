Arrays en Javascript

Para crear un array puedo usar indistintamente cualquiera de estas dos formas:

  • let meses = ['Enero', 'Febrero', ... , 'Diciembre'];
  • let meses = new Array('Enero', 'Febrero', ... , 'Diciembre');

Para acceder a los elementos de un array se accede especificando el índice del elemento, comenzando por el cero,

  • meses[1] => "Febrero"

Algunas funciones de los arrays:

  • Array.isArray( objetoSupuestoArray ) :: Boolean
  • .reverse( ) :: invierte el array (modifica el array original)
  • .sort( ) :: ordena alfabéticamente el array (modifica el array original)
  • .sort(funcionComparadora?: (a, b) => number) :: number[]
    Ordena un array numérico, debe usar una función comparadora (modifica el array original)
  • .concat( otroArray ) :: Array
    Append un array al final del array original.
    No modifica el array original, así que hay que recoger la operación en una variable
  • .indexOf( valorBuscado ) :: Number entero
  • .push( elementoNuevo ) :: Number entero (número de elementos en el array)
    Añade el/los nuevo(-s) elemento(-s) al final del array
  • .unshift( elementoNuevo ) :: Number entero (número de elementos en el array)
    Añade el nuevo elemento al inicio del array, desplazando a todos los demás
  • .pop() :: elemento eliminado
    Elimina el elemento que se halla al final del array
  • .shift() :: elemento eliminado
    Elimina el elemento que se halla al principio del array
  • .splice( index, numElementosAEliminar ) :: elemento(-s) eliminado(-s)
    Elimina el elemento que se halla al principio del array
