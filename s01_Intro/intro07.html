Tipos de dato en Javascript

Javascript es un lenguaje de tipo dinámico. El tipo de dato se guarda en el valor, no en la variable.
Se encuentran dos tipos de variables, las primitivas y las de referencia. Estas últimas son básicamente objetos.

  • string
  • numero
  • boolean
  • null
  • undefined
  • { ... objeto ... }

Con F12 podré ver ejercicios con el operador .typeof valor
